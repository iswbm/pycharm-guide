5.5 【绝佳工具 05】在Windows上使用 Bash 命令
============================================

|image0|

在 Windows 上的 cmd 命令和 Linux
命令有不少的差异，比如要列出当前目录下的所有文件，Windows 上是用 ``dir``
，而 Linux 上则是用 ``ls -l`` 。

对于像我这样熟悉 Linux 的开发者来说，Windows 的 那些 CMD
命令带来的糟糕体验是无法忍受的。
